# Data Directory

Ce dossier contient les fichiers JSON de configuration :

- `warehouse.json` : Configuration de l'entrepôt (zones, dimensions)
- `products.json` : Liste des produits (100 items)
- `agents.json` : Configuration des agents (robots, humains, chariots)
- `orders.json` : Commandes journalières à traiter

Ces fichiers seront fournis ou générés automatiquement.
